# cuddly-telegram

This repository contains a minimal Android app and a GitHub Actions workflow that builds a debug APK.

## Build and download APK from GitHub Actions

1. Open the **Actions** tab in GitHub.
2. Run **Android Build** (or push to a matching branch).
3. Open the workflow run.
4. Download the `app-debug-apk` artifact from the **Artifacts** section.

> Note: This project only provides a standard Android app build pipeline and does not include VPN or geo-restriction bypass functionality.
